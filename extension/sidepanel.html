Connect LinkedIn

Sign in to enable AI-powered outreach directly from LinkedIn profiles.

In LinkedIn Developers → your app → Auth → Authorized redirect URLs, add this URL exactly. Local dev uses 127.0.0.1 or localhost; production should use your real https://api… origin in REACHAI_PUBLIC_URL — same value you register on LinkedIn.

Unpacked extensions use a fixed ID until you remove them; if you re-load from another path, the ID can change — add the new URL in LinkedIn again.

We only request read access to help personalize your outreach.

Backend or AI unavailable
